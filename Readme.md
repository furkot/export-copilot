# furkot-copilot

Generate CoPilot .TRP files from [Furkot] trip data.

## Install

```sh
$ npm install --save furkot-copilot
```

## Usage

```js
var furkotCopilot = require('furkot-copilot');

furkotCopilot(trip);
```
